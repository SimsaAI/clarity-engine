Range Loops

Exclusive Range (1...10)

Numbers: {% for i in 1...10 %}{{ i }}{{ loop.last ? '' : ', ' }}{% endfor %}

Inclusive Range (1..10)

Numbers: {% for i in 1..10 %}{{ i }}{{ loop.last ? '' : ', ' }}{% endfor %}

Range with Step (0..100 step 10)

Numbers: {% for i in 0..100 step 10 %}{{ i }}{{ loop.last ? '' : ', ' }}{% endfor %}

Countdown (10..1 step -1)

Countdown: {% for i in 10..1 step -1 %}{{ i }}{{ loop.last ? '!' : ', ' }}{% endfor %}
